# clinical-lab-data-quality
Clinical laboratory data quality analysis - simulated dataset
## Overview
Simulated dataset analysis focused on data quality control in a clinical laboratory environment. This project applies SQL queries and data validation rules to detect anomalies, out-of-range results, and documentation inconsistencies.

## Skills demonstrated
- Clinical data validation and QC rules
- SQL queries for anomaly detection
- Laboratory data interpretation (hematology, biochemistry)
- Documentation and traceability (SOP-based approach)

## Dataset
Simulated clinical laboratory results (100 samples) including:
- Hematology panel (CBC)
- Basic biochemistry panel
- QC status and traceability fields
